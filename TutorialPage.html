Tutorials

Building construction

In the game you can construct buildings by right-clicking on an empty Plot and then entering the proper values into the field that pops up. The values are the following :

  • The name of the building
  • Choosing the type of the building (flat, hospital, school)
  • It`s area
  • The amount of turns it will take to build
Once the proper values have been entered you may start the building process. After waiting the amount of turns specified when placing the build, it will be completed.

Building repair

Building can be repaired by right-clicking them. This action will bring up the repair menu in which we must specify the cost of the project as well as how many turns we want it to take to complete. Afterwards similarly to the building process we have to wait the amount specified in the repair menu for the project to complete.

Game menu

From the hamburger menu in the left bottom corner of the screen the player can access 3 different dialogs. These 3 options are New service, End service and New project.

New service

With this option you can initiate a new service. A service can cover multiple buildings at once. When starting a new service you must enter the:

  • Name of the service
  • The type the service
  • The id of the building(s) you want the servuce to cover (separated by semicolons ;)
  • The cost of the service every turn
After finishing the creation process the service will now be live earning budget every turn as well as deducting its cost.

End service

When opening the end service dialog the list of all active services will be displayed along with their respective costs. After clicking the delete button located on the right side of a service entry the service will be deleted at which point it will stop costing draining the city`s budget.

New project

In the third and last menu the player can start projects. For a valid project 5 parametes must be set. These are in order :

  • The name of the project
  • The type of the project
  • The turns it will take to complete the project
  • The buildings included in the project (separated by semicolons ;)
  • The cost of the project for each turn
